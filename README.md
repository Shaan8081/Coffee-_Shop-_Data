# Coffee-_Shop-_Data
Coffee Shop Sales Analysis – Project Description  Analyzed a Coffee Shop Sales dataset to evaluate sales performance, customer purchasing behavior, and product trends. Cleaned and transformed the data, created key performance indicators (KPIs), and built an interactive dashboard to help monitor business performance .
 

Developed a comprehensive Coffee Shop Sales Analysis Dashboard using Microsoft Excel to analyze business performance and customer purchasing patterns. The project involved cleaning and transforming transactional sales data, creating dynamic reports, and visualizing key performance indicators (KPIs) to provide actionable business insights.

The dashboard enables stakeholders to monitor sales trends, evaluate product performance, identify peak business hours, and compare store performance across different locations. Interactive filters and visualizations allow users to explore data efficiently and support data-driven decision-making.

Key Features
Analyzed 120,000+ coffee shop sales transactions.
Cleaned and transformed raw data using Power Query.
Built an interactive dashboard with Pivot Tables, Pivot Charts, Slicers, and KPI Cards.
Tracked KPIs such as Total Revenue, Total Orders, Total Quantity Sold, Average Order Value, and Average Daily Sales.
Performed analysis by store location, product category, product type, month, day, and hour.
Identified best-selling products, top-performing stores, peak sales hours, and monthly sales trends.
Delivered business insights to support inventory planning, sales optimization, and strategic decision-making.
